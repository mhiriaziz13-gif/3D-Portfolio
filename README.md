# Ahmed Aziz Mhiri Portfolio

Public portfolio for Ahmed Aziz Mhiri, focused on data-driven marketing, commercial analytics, business intelligence, process automation and digital growth.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber and Drei

## Routes

- `/`
- `/about`
- `/projects`
- `/experience`
- `/resume`
- `/contact`

## CV Assets

Place CV files in `public/cv` using the exact names listed in `public/cv/README.md`, then mark the matching entries as available in `constants/portfolio.ts`.

## Commands

```bash
npm install
npm run type-check
npm run build
npm run lint
```

## Notes

This step only builds the public website design. CMS, Supabase, authentication, admin routes and contact-message persistence are intentionally not included yet.
