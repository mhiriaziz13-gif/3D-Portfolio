# AI search discoverability

The implementation improves machine-readable clarity through canonical URLs, server-rendered answer-style summaries, descriptive headings, Person/WebSite/CreativeWork/Breadcrumb structured data, consistent `sameAs` profiles and a concise `llms.txt`. Project descriptions remain first-hand and public-safe. `llms.txt` is supplementary and experimental; it is not an access-control file and does not guarantee AI retrieval or citation.

Governance rules: keep visible text and structured data aligned; show authorship and reliable dates only when known; update case studies when evidence changes; cite authoritative external sources in future insights; never add hidden summaries, fake citations, mass FAQs or keyword-variant pages.
