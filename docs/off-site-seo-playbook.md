# Off-site SEO playbook

Keep LinkedIn, GitHub and the portfolio consistent in name, headline, biography, location and canonical URL. Feature the portfolio and strongest case studies on LinkedIn; publish useful project lessons without invented results. Improve repository descriptions, topics and READMEs, and plan a GitHub profile README linking the portfolio.

Pursue only earned opportunities: legitimate alumni/credential profiles, approved employer or collaborator mentions, analytics/BI/hospitality-tech/UiPath/marketing communities, interviews and guest contributions built around useful diagrams or checklists. Never buy, exchange at scale, automate directory submissions, create fake profiles/reviews or imply endorsement.

Track referring domain, source URL, target URL, anchor, acquisition method, date, status, relevance and follow/sponsored/UGC classification.
